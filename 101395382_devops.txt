COMP 3104 is a DevOps course that focuses on software development automation, continuous integration (CI), and continuous deployment (CD).
COMP 3104 introduces students to DevOps principles and practices, emphasizing automation in the software development lifecycle.
Key topics include continuous integration, continuous deployment, automation, containerization, and monitoring.
Common tools used in DevOps include:
- Git for version control
- Jenkins for continuous integration
- Docker for containerization
- Kubernetes for orchestration
- Ansible for automation
